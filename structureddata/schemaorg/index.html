1. About the schema.org data set series

The Web Data Commons project has been extracting schema.org data from the Common Crawl every year since 2013 and offers the extracted data for public download in the form of the schema.org data set series. From a Web Science perspective, this data set series lays the foundation for analyzing the adoption process of schema.org annotations on the Web over the past decade. From a machine learning perspective, the annotations provide a large pool of training data for tasks such as product matching, product or job categorization, information extraction, or question answering. Webmasters primarily use the JSON-LD and Microdata formats to annotate structured data with schema.org, the extracted JSON-LD and Microdata corpora of the Web Data Commons Data Set are merged for a selection of schema.org classes to create class-specific subsets. The subsets contain all entities of a particular class together with entities of other classes that are found on the same pages
