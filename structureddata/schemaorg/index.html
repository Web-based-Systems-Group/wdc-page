Microdata, RDFa, JSON-LD, and Microformat

The Web Data Commons project has been extracting structured data from the Common Crawl every year since 2010. Since 2013, we are offering the schema.org data that we extract from the crawl for public download in the form of the schema.org data set series.

Contents

1. About the schema.org data set series

Webmasters primarily use the JSON-LD and Microdata formats to annotate structured data with schema.org, the extracted JSON-LD and Microdata corpora are merged for a selection of schema.org classes to create class-specific subsets. The subsets contain all entities of a particular class together with entities of other classes that are found on the same pages
