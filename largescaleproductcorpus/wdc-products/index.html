We offer the WDC Products benchmark for public download. The benchmark is currently available as a single zip file for each hardness level, 80%, 50% and 20% amount of hard negatives. Each zip file contains all corresponding development sets as well as test sets along the two remaining dimensions, amount of unseen products and development set size. Additionally the benchmark is available in two versions. One for pair-wise matching and another for multi-class matching.

The files are represented using the JSON format and can for example be easily processed using the pandas Python library.

import pandas as pd
df = pd.read_json('file_name.json.gz', compression='gzip', lines=True)

The code used for the creation of the corpus can be found on github.
